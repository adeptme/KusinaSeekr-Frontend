# KusinaSeekr
**A Web-Based Filipino Recipe and Learning Platform for Promoting Home Cooking and Culinary Education**

---

## Description
**KusinaSeekr** is a web-based platform designed to promote and encourage home cooking and culinary education.  
It offers a convenient way for users to find, learn, and share Filipino recipes, blending cultural appreciation with modern technology.

---

## Objectives

This project aims to make cooking easier and more accessible by allowing users to explore and learn Filipino dishes interactively.

### Platform Goals
To design and develop a web-based Filipino recipe and learning platform with:
- Recipe and ingredient-based search system
- Step-by-step cooking procedures with educational resources
- User reviews and feedback on recipes
- Community engagement through skill sharing and discussions

- Providing personalized recipe suggestions based on available ingredients
